# RAG pipeline for bank chat-bot question answering

### Описание
В проекте представлен RAG пайплайн для автоматизации ответов на вопросы клиентов банковского сервиса. Он основывается на собранныз данных со всех страниц сайта банка. Был создан в рамках хакатона Альфа-Будущее.

### Структура
Проект состоит из ретривера `hybrid_search1.py`, который реализует гибридный (dense & bm25plus) поиск по индексированным чанкам и возвращает самые релевантные совпадения. 

Реранк использовать *не рекомендуется*, так как он может привести к ухудшению качества ответов.

В классе `rag_generator.py` реализован ретрив релевантных чанков, создание user и system промптов и запрос к серверу модели.

Взаимодействие и интерфейс реализованы через Gradio.

### Запуск
Для запуска необходимо установить необходимые зависимости:
```bash
pip install -r requirements.txt
```

Далее запустить с указанием OpenAI API compatible сервера LLM:
```bash
python main.py --model_name <model_name> --host 0.0.0.0 --port 8000
```

После запуска будет развернут Gradio интерфейс на `http://127.0.0.1:7860`
